# soralog

Ecommerce catalog inspired by Sorabel

## Tools

### Web (Catalog Site)

Show catalog and can see the details

```bash
cd web
npm install
npm start
```

### Dashboard (Admin)

CRUD for products

```bash
cd dashboard
npm install
npm start
```

### Backend

Serve data for frontend via GraphQL and RestAPI

```bash
cd server
npm install
npm start
```
